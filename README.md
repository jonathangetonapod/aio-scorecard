# AIO Readiness Scorecard

B2B lead generation tool that shows prospects their AI visibility gap.

## Features

- 🔍 Queries GPT-5 + Perplexity with customer-intent questions
- 🏢 Auto-detects company, services, and location from domain
- 📊 Shows who AI recommends instead (competitors)
- 💰 Calculates potential revenue impact
- 📄 Exports branded PDF reports

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Set up environment
cp .env.example .env
# Add your API keys

# Run the API
python3 api.py
```

Server runs at http://localhost:3002

## Tech Stack

- Python (FastAPI)
- OpenAI GPT-5
- Perplexity API
- PDF generation
