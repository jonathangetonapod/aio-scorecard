# Client Feedback - David Zybin (MarketMagnetix) - Feb 24, 2026

## Key Changes Needed

### 1. Target Audience Refinement
- **Only 50-200 employees** (sweet spot)
- Big enough to have budget
- Not competing with giants like Lockheed Martin, L3 Harris, GE

### 2. Shift from Location → Vertical-Based Queries
**Before:** "Best aerospace companies in Arvada, Colorado"
**After:** Analyze each manufacturer's website, extract their specific vertical/keywords, then query LLMs for that vertical

**New Flow:**
1. Scrape manufacturer's website
2. Extract most prominent/unique keywords identifying their vertical (e.g., "investment casting", "turbine blades", "molding")
3. Run LLM queries based on vertical: "find me [casting] manufacturing companies"
4. Compare against similar competitors in that specific vertical
5. Show where they rank vs vertical competitors (not geographic competitors)

### 4. Email Personalization
Use extracted vertical in email copy:
> "I noticed you're not ranking in LLMs for what you focus on — **[molding]**. I ran a report comparing you to other manufacturers that focus on **[molding]**, and this is where you rank. Can I send over the report?"

## Technical Implementation

### Website Vertical Extraction
- Analyze website content
- Extract most prominent unique keywords
- Map to manufacturing vertical
- Use those keywords for LLM queries instead of location

### Query Format Change
**Old:** "What are the best {industry} companies in {city}, {state}?"
**New:** "Find me {extracted_keyword} manufacturing companies" or "Best {vertical_keyword} manufacturers"
